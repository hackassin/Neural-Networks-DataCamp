This is a custom package/library for code reusability
Modules added so far:
1. ecdf_compute: Computes ecdf for a set of data
2. diff_means: Computes difference of means between two types of data
3. permutation_repl: Computes permutation replicates based on the test statistic provided
4. pearson_coeff: Computes pearson_r correlation factor between two types of data
5. bootstrap_repl: Computes bootstrap replicates based on the test statistic provided
6. finch_beaks_df: Extracts dataframes from the Finch beaks dataset
